# To-Do List Desktop App

A feature-rich task management desktop application built with Python and Tkinter. Features a modern dark UI with persistent local storage.

## Features
- ✅ Add, complete, and delete tasks
- 🎨 Modern dark theme UI (Catppuccin Mocha)
- 💾 Persistent storage — tasks saved in `tasks.json`
- 🔍 Filter tasks: All / Pending / Completed
- 🗑️ Clear all completed tasks at once
- 📊 Live stats (total, done, pending)
- ⌨️ Press **Enter** to add task quickly

## Tech Stack
- Python 3.x
- Tkinter (built-in GUI library)
- JSON (file-based storage)

## How to Run

```bash
python main.py
```

No external libraries needed — 100% Python standard library.
